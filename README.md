# Scheasy v2.0 💈

> **Sistema Profissional de Gestão e Agendamento para Barbearias Modernas.**

O **Scheasy** é uma solução completa para automatizar o fluxo de agendamentos, melhorar a experiência do cliente e profissionalizar a gestão de barbearias. O sistema permite que clientes agendem serviços online com sincronização em tempo real e oferece aos administradores ferramentas robustas de controle.

---

## 🚀 Propósito do Projeto

Eliminar as barreiras entre o cliente e o barbeiro, substituindo agendamentos manuais via mensagens por uma plataforma automatizada 24/7. O foco é permitir que o profissional foque no seu ofício enquanto o sistema cuida da logística de horários e lembretes.

## 📈 Benefícios para o seu Negócio

*   **Disponibilidade 24/7:** Receba agendamentos mesmo quando a barbearia está fechada.
*   **Redução de Faltas (No-show):** Lembretes automáticos (WhatsApp/Email) para os clientes.
*   **Sincronização com Google Agenda:** Evite conflitos de horários e gerencie sua rotina direto do celular.
*   **Fidelização de Clientes:** Cadastro organizado e histórico de serviços realizados.
*   **Profissionalismo:** Uma interface moderna que agrega valor à marca da sua barbearia.

---

## 🛠 Tecnologias Utilizadas

### Frontend (Interface do Usuário)
*   **React 19:** Biblioteca principal para construção da interface.
*   **TypeScript:** Garantia de segurança e robustez no código.
*   **Vite:** Ferramenta de build ultra-rápida.
*   **Tailwind CSS:** Estilização moderna e responsiva.
*   **Lucide React:** Conjunto de ícones profissionais.
*   **React Router Dom:** Gerenciamento de navegação entre páginas.

### Backend (Servidor e Lógica)
*   **Node.js & Express:** Ambiente de execução e framework para a API.
*   **Prisma ORM:** Ferramenta avançada para manipulação do banco de dados.
*   **SQLite:** Banco de dados ágil e eficiente para o ambiente de desenvolvimento.
*   **JWT (JSON Web Token):** Autenticação segura para administradores e clientes.
*   **BCryptJS:** Criptografia de alta segurança para senhas.

### Integrações de Terceiros
*   **Google Calendar API:** Sincronização automática de agendamentos.
*   **Twilio API:** Envio de notificações e lembretes via WhatsApp.
*   **Google OAuth 2.0:** Login simplificado e seguro para usuários.

---

## ⚙️ Como Executar o Projeto

1.  Clone o repositório.
2.  Certifique-se de ter o **Node.js** instalado.
3.  Simplesmente execute o arquivo `iniciar_projeto.bat` na raiz do projeto.
    *   O script irá instalar dependências, configurar o banco de dados e abrir o sistema no seu navegador automaticamente.
